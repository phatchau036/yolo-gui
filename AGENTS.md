# AGENTS.md

Thường xuyên viết các file `.md` để dev sau dễ dàng tiếp quản dự án. Tài liệu cần bao gồm:

- Phong cách đặt biến.
- Phong cách code.
- Phong cách xử lý logic.
- Mô tả logic của dự án.
- Todo list cần làm.
- Việc đã hoàn thành.
- Cách fix bug.
- Chỗ ghi log.
- Memory dự án.

Khi sửa code, cập nhật tài liệu liên quan trong `docs/` cùng lượt thay đổi.
